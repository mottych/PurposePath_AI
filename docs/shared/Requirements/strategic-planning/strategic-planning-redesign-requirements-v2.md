## 1. Executive Summary

### Vision

Transform the Strategic Planning experience by separating two distinct user mindsets:

- **Planning Mode** (Strategic Planning section): Deliberate, wizard-guided goal creation
- **Tracking Mode** (Operations section): Rapid, dashboard-focused execution monitoring

### Key Deliverables

1. **Planning Studio** - Wizard-based goal creation with 5 guided steps
2. **Goals Overview** - Visual goal portfolio with status cards
3. **Command Center** - Operations dashboard showing what needs attention
4. **Measure Tracker** - Centralized performance monitoring with quick entry
5. **Action Board** - Task management with multiple view options
6. **Personal Scorecard** - Individual Measure assignments and tracking
7. **Goal Drill-Down** - Read-focused deep view via slide-out panels

### Success Metrics

- Reduce time to create a complete goal from 15+ minutes to <8 minutes
- Increase daily Measure update frequency by 40%
- Reduce clicks to record an actual from 5 to 2
- Achieve 90%+ user preference for new experience

## 2. Design Philosophy

### Core Principles

| Principle | Implementation |
|-----------|----------------|
| **Mode Separation** | Planning vs Tracking have distinct UIs, colors, and workflows |
| **Progressive Disclosure** | Summary first, expand for details, slide-out for context |
| **Action-Oriented** | Every screen answers "What should I do next?" |
| **AI as Assistant** | Suggestions offered, never forced; reasoning always shown |
| **Minimal Friction** | Quick entry everywhere; inline editing; keyboard shortcuts |

### UI Patterns

| Pattern | Usage |
|---------|-------|
| **Slide-Out Panels** | Context viewing, quick edits, detail drill-down (from RIGHT) |
| **Expanding Cards** | Strategy details, Measure details, action details |
| **Wizard Steps** | Planning Studio goal creation |
| **Status Pills** | Quick visual scanning of state |
| **Progress Rings** | Compact progress visualization |
| **Inline Quick Entry** | Recording actuals, updating progress |

### Color Language

| Color | Meaning | CSS Variable |
|-------|---------|--------------|
| **Emerald** (`#10B981`) | On Track, Success, Healthy | `--status-success` |
| **Amber** (`#F59E0B`) | At Risk, Warning, Attention | `--status-warning` |
| **Rose** (`#F43F5E`) | Behind, Critical, Blocked | `--status-danger` |
| **Blue** (`#3B82F6`) | Planning Mode, Primary Actions | `--mode-planning` |
| **Teal** (`#14B8A6`) | Operations Mode, Tracking | `--mode-operations` |
| **Slate** (`#64748B`) | Neutral, Draft, Inactive | `--status-neutral` |
