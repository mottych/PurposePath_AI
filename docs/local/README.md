# Local Docs Boundary

## Purpose

`docs/local/` contains repository-specific documentation for `PurposePath_AI` that should not be synchronized to other repositories.

## What Belongs Here

- Python/FastAPI architecture and implementation standards.
- AI service coding and testing standards.
- Local deployment, observability, and production runbook references.
- Local process checklists and operations runbooks.
