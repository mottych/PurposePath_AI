# Local Archive Boundary

`docs/local/archive/` contains historical repository-specific documents retained for traceability.

Archived docs are not canonical standards.

## Subfolders

- `guides-legacy/`: Previous guide sets replaced by canonical local guides.
- `status/`: Historical status snapshots and execution summaries.
- `issues/`: Historical issue analysis and action plans.
- `analysis/`: Historical architecture or implementation assessments.
- `migration/`: One-time restructuring and migration notes.

For current implementation rules, use:

- `docs/local/guides/architecture-standards.md`
- `docs/local/guides/coding-standards.md`
- `docs/local/guides/development-guidelines.md`
